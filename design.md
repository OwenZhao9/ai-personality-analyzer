# AI 多模型联合人格分析系统 - 设计方案

## 应用概述

深色科技风格的 AI 人格分析应用，以 `#0A0E1A` 深蓝黑为主背景，`#00D4FF` 青蓝为主色调，营造高科技感。

## 色彩体系

| 用途 | 颜色值 |
|------|--------|
| 主背景 | `#0A0E1A` |
| 主色调（青蓝） | `#00D4FF` |
| 副色调（紫） | `#A78BFA` |
| 主文字 | `#E8EEF4` |
| 次要文字 | `#7A8FA6` |
| 暗淡文字 | `#4A5568` |
| 卡片背景 | `rgba(255,255,255,0.04)` |
| 边框 | `rgba(255,255,255,0.08)` |

## 屏幕列表

### 1. 欢迎页（Welcome Screen）`/(tabs)/index`
- 深色背景 + 左右两侧飘动的 AI 模型名称标签
- 中央：脉冲光环 + AI Logo 圆圈
- 标题：「AI 多模型联合 / 人格分析系统」
- 副标题 + 「开始分析」按钮（青蓝色圆角）
- 底部免责声明

### 2. 信息输入页（Input Screen）`/input`
- 顶部返回按钮 + 标题
- 基础信息：姓名（必填）、性别选择、出生日期（滚轮弹窗）
- 社会角色：职业、兴趣爱好、情感状态
- 人格深度：MBTI、当前困惑、一句话自我介绍
- 底部「启动多模型分析」按钮

### 3. 分析动画页（Analysis Screen）`/analysis`
- 全屏深色背景 + 扫描线动画
- 分阶段展示：数据处理 → 模型并行分析 → 共识校验
- 24 个 AI 模型卡片逐个出现
- 进度条 + 百分比数字动画
- 完成后自动跳转揭示页

### 4. 揭示过渡页（Reveal Screen）`/reveal`
- 第一页：「即将揭示你的深层人格核心」+ 三个跳动点
- 第二页：核心揭示文案 + 「查看完整报告」按钮

### 5. 报告页（Report Screen）`/report`
- 顶部：用户基础信息卡片（姓名、年龄、星座等）
- 各维度分析报告（基于输入生成）
- 底部固定：「生成海报并分享」按钮
- 海报预览弹窗 + 保存/分享功能

## 关键用户流程

1. 欢迎页 → 点击「开始分析」→ 信息输入页
2. 信息输入页 → 填写信息 → 点击「启动多模型分析」→ 分析动画页
3. 分析动画页（约 30 秒动画）→ 自动跳转揭示页
4. 揭示页 → 点击「查看完整报告」→ 报告页
5. 报告页 → 点击「生成海报并分享」→ 海报预览弹窗 → 保存/分享

## iOS 发布功能

在报告页底部区域添加「发布到 iOS App Store」入口按钮，引导用户了解如何将应用发布到 iOS。
